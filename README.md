## About This Project

This is a **Telegram Bot application** built with Laravel, designed to provide automated bot functionality through the Telegram messaging platform. The application leverages Laravel's powerful features combined with the Telegram Bot SDK to create a robust and scalable bot solution.

## Features

- **Telegram Bot Integration** - Full integration with Telegram Bot API
- **Command Processing** - Support for various bot commands and interactions
- **Database Queue System** - Efficient job processing using MySQL database queue
- **Real-time Processing** - Handle incoming messages and webhooks
- **Logging & Monitoring** - Comprehensive logging with Monolog
- **Cron Job Support** - Scheduled tasks using Laravel's task scheduler

## Technology Stack

- **Framework:** Laravel 12.36.1
- **PHP:** 8.2+
- **Database:** MySQL
- **Queue:** Database-driven queue system
- **Bot SDK:** irazasyed/telegram-bot-sdk v3.15.0
- **Frontend:** Vite + Tailwind CSS 4.0
- **HTTP Client:** Guzzle HTTP 7.10.0
